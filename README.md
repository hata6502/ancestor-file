> A CLI to find a file from the ancestor directories.

## Install

```bash
npm install -g ancestor-file
```

## Usage

```bash
$ cd ~/example
$ ls -a
.  ..  .env  directory
$ cd directory
$ anc .env
/home/user/example/.env
```
